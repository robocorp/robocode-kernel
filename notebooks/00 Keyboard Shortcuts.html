Navigating cells

up / j

select cell above

down / k

select cell below

y

change cell to code mode

m

change cell to markdown

enter

enter edit mode

esc

exit edit mode

Managing cells

a

insert cell above

b

insert cell below

c

copy cell

v

paste cell

d,d

delete dell

shift + (up / j)

extend selection above

shift + (down / k)

extend selection below

shift + m

merge selected cells

Executing cells

ctrl + enter

run cell

shift + enter

run cell, select below

alt + enter

run cell, insert below

Edit mode

tab

suggest code completion

ctrl + shift + -

split cell in half

esc

exit edit mode
